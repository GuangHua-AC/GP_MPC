# Common scripts

通用流程脚本，适用于 `balance`、`balance_turn`、`balance_turn_roll`、`height`、`terrain` 等任务。

```text
01_run_pd.py       运行 PDF 动力学 + PD 控制
02_collect_data.py 采集模型训练数据
03_train_nn.py     训练 NN 动力学模型
04_run_nn_mpc.py   运行 NN + MPC
05_train_gp.py     训练 GP 动力学模型
06_run_gp_mpc.py   运行 GP + MPC
```
