## Current Pure Python Scope

The current closeout scope contains five pure Python GP-PMPC tasks:

1. `balance`
2. `turn`
3. `height`
4. `known terrain`
5. `terrain adaptive`

Jump is kept as a smoke / exploratory module and is not counted as one of the five main GP-PMPC tasks.

## Task Goals

| Task | Goal |
|---|---|
| balance | Maintain stable `theta` and `phi` under a 30 N / 120 ms external push. |
| turn | Track `yaw_ref=30 deg` with `v_ref=0.15`, while suppressing roll. |
| height | Track fixed / step / sine `L0` references while keeping `theta` and `phi` stable and keeping `x` bounded. |
| known terrain | Track `leg_diff` under a known left/right terrain height difference, suppress roll, and keep the body balanced. |
| terrain adaptive | Without directly using known terrain height difference, use a blind adaptive wrapper to adjust `leg_diff` and maintain stability under terrain feedback. |
